### UserInfo

| Field | Type | Label | Description |
| ----- | ---- | ----- | ----------- |
| uid | [string](#string) |  | 用户在鉴权网关中的唯一ID |

| avatar | [string](#string) |  | 用户头像路径 |

| IAM | 1 | IAM系统 |
